# 🎨file_trans 前端服务

一个基于 **Vue 3** 构建的现代化单页应用（SPA），集成 **深色 / 浅色主题切换** 与 **文件上传进度模拟** 功能，采用组合式 API 和清晰的项目结构，适合作为前端工程模板或示例项目。

------

## ✨ 项目简介

本项目基于 **Vue 3 + Composition API（<script setup>）** 构建，支持系统主题自适应与用户手动切换，并通过组合式函数模拟文件上传流程，实现上传进度的可视化管理。

项目遵循 Vue 官方最佳实践，结构清晰，易于维护与扩展。

------

🧰 技术栈 

| 分类     | 技术                               |
| -------- | ---------------------------------- |
| 核心框架 | Vue 3                              |
| 语法风格 | Composition API (`<script setup>`) |
| 路由管理 | Vue Router                         |
| 样式方案 | CSS Variables（CSS 自定义属性）    |
| 状态存储 | localStorage                       |
| 架构模式 | Composables                        |

------

## 🚀 核心功能

### 1️⃣ 主题切换系统

- 🌗 **系统主题自适应**
  - 初始化自动检测系统深色 / 浅色偏好
  - 用户选择持久化到 `localStorage`
- 🎨 **CSS 变量主题系统**
  - 使用 `data-theme` 属性控制主题
  - 避免组件中硬编码颜色
- ✨ **视觉效果**
  - 0.5s 平滑颜色过渡动画
  - 渐变背景与网格动效
  - 太阳 / 月亮图标状态反馈

------

### 2️⃣ 文件上传模拟功能

基于组合式函数 `useFileUpload.js` 实现。

**功能特性：**

- 📈 模拟上传进度（每 500ms 更新）
- 🧮 计算整体上传进度
- 🔄 实时同步单文件与全局状态
- 🧹 自动清理定时器，避免内存泄漏

> 当前为模拟实现，生产环境可直接替换为真实上传接口。

------

📁 项目结构 

```
src/
├── App.vue               # 根组件（主题切换 + 路由出口）
├── main.js               # 应用入口
├── composables/
│   └── useFileUpload.js  # 文件上传逻辑封装
├── components/           # 通用组件
├── views/                # 页面视图
├── stores/               # 状态管理（可扩展 Pinia）
├── utils/                # 工具函数
├── router/               # 路由配置
└── assets/               # 静态资源
```

## 🛠️ 开发指南

### 🎨 新增主题

在 `App.vue` 中扩展 CSS 变量即可：

```css
:root[data-theme="high-contrast"] {
  --bg-primary: #000000;
  --text-color: #ffffff;
}
```

------

### 📦 扩展文件上传功能

- 在 `useFileUpload.js` 中支持多文件上传
- 使用 `axios + FormData` 替换模拟逻辑，对接真实后端接口

------
